# Exercise 1: Basic Git Workflow

This exercise demonstrates the basic Git operations using GitHub.

## Steps Performed

1. Created a new GitHub repository.
2. Created a README file.
3. Made the first commit.
4. Edited the README file and committed the changes again.
5. Viewed the commit history to verify changes.

## Outcome

Successfully learned how to:
- Create files
- Commit changes
- Track history in GitHub
